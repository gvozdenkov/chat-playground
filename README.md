# Chat App Sandbox

Test chat creation with `Nestjs` backend, and `React` frontend

## Local Dev

```bash
git clone git@github.com:gvozdenkov/websocket-chat.git

# use node 20

# start backend

cd websocket-chat/backend

npm i

npm run start:dev

# start frontend
cd ../frontend

npm i

npm run dev

# open 2 browser windows and open http://localhost:5173/
# Start chating :)
```
